계산기 사용 가이드

목차

    세금 계산기

    취득세 양도소득세 종합소득세 연말정산 종합부동산세 재산세 증여세 상속세 보유세 통합 임대소득세
    주의 세율과 공제는 매년 개정되므로 세법 히스토리에서 최신 반영일을 확인하세요.

    대출 계산기

    대출이자 DSR DTI LTV 대출가능액

    비용 계산기

    중개보수 등기비용 법무사수수료 인지세 채권할인료 감정평가수수료

    생활 계산기

    임대수익률 공동명의 평수변환 용적률·건폐율 경매비용 리모델링 수익 건물 잔존가치

    PRO 분석

    총비용 시뮬레이터 세금비교 분석 투자수익 분석

    상황별 추천

    집 살 때

    취득세 → 등기비용 → 대출가능액 → 총비용 시뮬레이터 순서로 확인하세요.

    취득세 → 등기비용 → 대출가능액 →

    집 팔 때

    양도소득세 + 중개보수로 예상 실수익을 계산하세요.

    양도소득세 → 중개보수 →

    대출 받을 때

    DSR → LTV → 대출이자 순서로 한도와 월 상환액을 확인하세요.

    DSR → LTV → 대출이자 →

    임대할 때

    임대수익률 → 임대소득세 → 전월세전환 순서로 수익을 검증하세요.

    임대수익률 → 임대소득세 → 전월세전환 →

    경매할 때

    경매비용 → 취득세 → 투자수익분석 순서로 낙찰가 상한을 산출하세요.

    경매비용 → 투자수익 분석 →

    계산기 사용 시 주의사항

    • 공시가격 vs 시가 — 재산세·종부세는 공시가, 양도세는 실거래가 기준
    • 주택수 산정 — 분양권·입주권 포함 여부를 계산기 옵션에서 확인
    • 지방소득세 — 양도세·종합소득세의 10% 자동 가산
    • DSR 대출 포함 — 신용대출·카드론·할부까지 전부 입력
    • 최종 확정 전 — 세무사·법무사 상담으로 엣지케이스 확인
