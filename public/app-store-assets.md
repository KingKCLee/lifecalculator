# App Store 등록 에셋 체크리스트 — 생활계산기

## 1. 앱 아이콘

App Store Connect 업로드용 1024x1024 단일 PNG만 있으면 Xcode/Capacitor가 나머지 사이즈를 자동 생성합니다.

| 용도 | 사이즈 (px) | 형식 |
|---|---|---|
| App Store 마케팅 | **1024×1024** | PNG, 알파 채널 없음, sRGB 또는 P3 |
| iPhone (앱) | 180×180, 120×120 | PNG |
| iPhone Spotlight | 120×120, 80×80 | PNG |
| iPhone Settings | 87×87, 58×58 | PNG |
| iPhone Notification | 60×60, 40×40 | PNG |
| iPad (앱) | 167×167, 152×152 | PNG |
| iPad Spotlight | 80×80, 40×40 | PNG |
| iPad Settings | 58×58, 29×29 | PNG |

### 자동 생성

```bash
npm install -D @capacitor/assets
# assets/icon.png (1024x1024) 준비 후
npx capacitor-assets generate --ios \
  --iconBackgroundColor "#0747A6" \
  --iconBackgroundColorDark "#0747A6"
```

### 디자인 가이드
- 배경 단색 권장 (생활계산기 브랜드 컬러 `#0747A6`)
- 중앙에 흰색 계산기 아이콘 또는 "₩" 심볼
- 모서리 라운딩은 iOS가 자동 적용 — **직사각형 풀 캔버스로 제작**
- 텍스트는 가독성 어려우므로 심볼 위주

## 2. 스크린샷

App Store는 **6.7" iPhone**과 **6.5" iPhone**, **5.5" iPhone**, **12.9" iPad** 카테고리별로 별도 업로드 필요. 한 카테고리당 최소 3장, 최대 10장.

| 디바이스 | 사이즈 (px, Portrait) | 비고 |
|---|---|---|
| **iPhone 6.7"** (15 Pro Max, 14 Pro Max, 14 Plus 등) | 1290×2796 | **필수** |
| **iPhone 6.5"** (11 Pro Max, XS Max) | 1242×2688 또는 1284×2778 | **필수** |
| **iPhone 5.5"** (8 Plus) | 1242×2208 | **필수** |
| **iPad Pro 12.9"** (3세대~) | 2048×2732 | iPad 지원 시 필수 |
| **iPad Pro 11"** (3세대~) | 1668×2388 | 선택 |

> 6.7" 스크린샷만 업로드해도 6.5"·5.5"는 자동 스케일링 가능 (App Store Connect 옵션). 하지만 품질을 위해 **6.7"·6.5" 두 종은 직접 제작 권장**.

### 스크린샷 구성 (10장 권장 순서)

1. **메인 화면** — "40개 무료 계산기" 카피 + 카테고리 그리드
2. **취득세 계산기** — 입력 → 결과 화면
3. **양도소득세 계산기** — 결과 + 절세팁
4. **연봉 실수령액** — 결과 그래프
5. **DSR 계산기** — 한도 비교
6. **부동산 용어집** — 검색 화면
7. **마이페이지** — 히스토리·즐겨찾기
8. **AI 해설** — Claude 결과 박스
9. **공인중개사 브랜드 PDF** — 발급 화면
10. **다크모드/접근성** — 큰 글자 옵션 (있다면)

각 스크린샷 상단에 **굵은 한 줄 카피** 권장 (예: "한 번에 계산, 즉시 결과"). Figma·Canva·Screenshots Builder 등으로 후가공.

## 3. 앱 미리보기 영상 (선택)

- 길이: 15~30초
- 형식: H.264, MP4 또는 MOV
- 사이즈: 스크린샷과 동일 해상도
- 내용 권장: 메인 → 계산기 진입 → 입력 → 결과 → 공유 → 마이페이지 (5컷)
