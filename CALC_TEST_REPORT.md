## 항목별 상세

### 1. 경매 적정 입찰가 (CalcAuction2)
- **입력:** 감정가(원), 소재지(수도권/지방), 유찰 횟수, 권리분석 난이도
- **계산 로직:**
  - 최저매각가 = 감정가 × (1 − step)^유찰횟수 (metro 0.2, other 0.3)
  - 권장입찰가 = 감정가 × 낙찰가율(0.83/0.72) × (1 − 리스크할인)
- **검증:** 감정가 80000만원·수도권·1회 유찰·보통 → 평균 낙찰가 6.64억, 권장 6.308억 (수식 확인)
- **런타임 확인 필요:** 유찰 횟수 음수/문자 입력 시 `Math.max(0, parseInt)` fallback 확인

### 2. 경매 배당표 (CalcAuctionDiv)
- **입력:** 매각대금, 집행비용, 소액임차 보증금, 해당지역, 당해세, 근저당1/2, 일반채권
- **계산 로직:** 배당 순위 1~5 순차 차감 후 잔액
- **특이사항:** 8개 state + 지역별 최우선변제 한도 분기
- **런타임 확인 필요:** 지역 선택 변경 시 최우선변제 한도가 즉시 반영되는지

### 3. 재건축 가능연한 (CalcReconYear)
- **입력:** 준공연도, 소재지, 구조(RC/벽식), 안전진단 여부
- **계산 로직:** 기준연한 − (현재년 − 준공년); 안전진단 통과 시 연한 무시
- **검증:** 1995년·서울(30년)·2026년 현재 → 경과 31년, 신청 가능
- **런타임 확인 필요:** 미래 년도 입력 시(예: 2030) 음수 경과 처리

### 4. 리모델링 타당성2 (CalcRemodel2)
- **입력:** 현재시세, 예상가, 공사비, 대출금, 금리, 공사기간, 이사비, 임시거주비
- **계산 로직:** 순이익 = 차익 − 공사비 − 이자(공사기간) − 기타비용 − 양도세 추정
- **런타임 확인 필요:** 대출금 > 공사비일 때 음수 이자 처리

### 5. 임대료 5% 상한 (CalcRentIncrease)
- **입력:** 기존 보증금·월세, 전환율, 인상 희망, 신규 보증금
- **계산 로직:** 월세 환산 → 5% 상한 검증 → 위반 여부 판정
- **런타임 확인 필요:** 전환율 0 입력 시 division-by-zero 방어

### 6. 경락잔금대출 (CalcAuctionLoan)
- **입력:** 낙찰가·감정가·소재지·지역구분·방공제·대출기간·금리
- **계산 로직:** 한도 = min(낙찰×LTV, 감정×LTV×0.9) − 방공제
- **런타임 확인 필요:** 방공제 > 한도 시 0 또는 음수 처리

### 7. 대환대출 비교 (CalcRefinance)
- **입력:** 원금·기존금리·신규금리·잔여기간·중도상환수수료율·거치·이동비용
- **계산 로직:** 월 상환액 amortMonthly 함수 사용, 총이자 차이 = 신·구 비교
- **런타임 확인 필요:** amortMonthly 내부 r≤0 가드 존재 (확인 완료)

### 8. 건물분 부가세 (CalcBldVat)
- **입력:** 총 매매가·토지 비율·주거용 여부·전용면적·과세 유형
- **계산 로직:** 건물가 = 매매가 × (1 − 토지비율); VAT = 건물가 × 10/110
- **런타임 확인 필요:** 85㎡ 이하 주거용 → VAT 면세 분기

### 9. 임대 추정소득 (CalcEstIncome)
- **입력:** 월 임대료, 보증금, 간주임대료 포함, 필요경비율
- **계산 로직:** 연 수입 × (1 − 경비율) = 소득금액
- **런타임 확인 필요:** 경비율 100% 초과 입력 방지

### 10. 착한임대인 세액공제 (CalcGoodLandlord)
- **입력:** 기존/인하 월 임대료, 개월수, 종합소득 1억 초과 여부
- **계산 로직:** 인하액 × 개월 × (70% or 50%)
- **런타임 확인 필요:** 인하액이 음수(인상)일 때 0 처리

### 11. 간주임대료 (CalcImputedRent)
- **입력:** 보증금 합계, 주택수, 정기예금이자율, 기간
- **계산 로직:** (합계 − 3억) × 60% × 이율; 3주택 미만이면 0
- **런타임 확인 필요:** 보증금 3억 미만 시 0 반환

### 12. 법정상속분 (CalcLegalInherit)
- **입력:** 상속재산·배우자 여부·자녀수·직계존속수
- **계산 로직:** 배우자 1.5 / 자녀 1 비율 분모 합산
- **검증:** 10억·배우자+자녀2 → 배우자 4.286억, 자녀 각 2.857억 (수식 확인)
- **런타임 확인 필요:** 배우자 없이 자녀 0·부모 0 → 0 분배 처리

### 13. 누진세 범용 (CalcProgressive)
- **입력:** 세목(종합소득/양도/증여/상속/종부세), 과세표준
- **계산 로직:** `pTx(base, table)` 누진세 헬퍼 호출
- **런타임 확인 필요:** 한계세율·실효세율 소수점 표시

### 14. 국민주택채권 매입 (CalcBond2)
- **입력:** 공시가격·매입요율·할인손실률·토지/주택 구분
- **계산 로직:** 공시가 × 매입률 = 액면, 할인손실 = 액면 × 할인률
- **런타임 확인 필요:** 공시가격 2천만 이하 면제 구간 분기

### 15. 전세보증보험료 (CalcJeonseInsurance)
- **입력:** 보증금·주택유형·보증기간·저소득 할인·청년 할인
- **계산 로직:** 요율 × 보증금 × (개월/12) × 할인
- **런타임 확인 필요:** 할인 중복 적용 한도(40%) 상한

### 16. 인지세 (CalcStamp)
- **입력:** 거래금액·전자계약 여부·매수인 부담비율·거래유형
- **계산 로직:** 구간별 고정액 (1억이하 비과세 ~ 10억초과 35만원), 전자계약 50% 할인
- **런타임 확인 필요:** 전자계약 할인 적용 후 천원 단위 반올림

### 17. 날짜 계산기 (CalcDateDiff)
- **입력:** 시작일·종료일(YYYYMMDD), 모드(기간/Dday), 기준일
- **계산 로직:** Date 파싱 → (end-start)/(1000*60*60*24)
- **런타임 확인 필요:** 잘못된 날짜 포맷 입력 시 안내 메시지

### 18. 손없는 날 달력 (CalcLuckyDay)
- **입력:** 기준 월·달력 모드
- **계산 로직:** Intl.DateTimeFormat chinese 캘린더로 양→음력 변환, 음력 9/19/29일 하이라이트
- **특이사항:** `useMemo` + `useEffect` 2개 사용 (다른 계산기 대비 복잡)
- **런타임 확인 필요:** 구형 브라우저(chrome < 80) Intl chinese 지원 여부

정적 분석으로는 놓치기 쉬운 케이스:

5. **접근성** — label/aria 속성, 키보드 Tab 순서
